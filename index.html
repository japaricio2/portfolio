Selected Projects

Lyriqz

Live Github

Lyriqz is built for the person that wants to sing along but just can't remember the words. By using the Spotify API I'm able to find out what you're currently listening to and fetch those lyrics for you. All you ever have to do anymore is refresh the browser when a new song starts to play. In the future, I'll definitely be adding auto-refresh to the project.

HelmetBot

Live Github

HelmetBot is a web service that makes searching Yelp from within your Telegram chats super easy. This web service integrates with a Telegram bot I created to make searching and sharing a lot less cumbersome. Check out the live site for a demo of it in action and more details.

Quizzy

Live Github

Quizzy is a web application that allows you to create custom quizzes. Once you're done creating a quiz, easily share it with your friends. It was created to send random and funny quizzes to test your friends. In the future I'll be adding more features to make this more of an educational tool.
